# AI Social Automation

Web app per creare grafiche social Italian Wine Academy con layout canvas preciso, campi modificabili e download PNG.

## Funzioni

- Template 1080x1350 per post Instagram
- Campi modificabili da prompt
- Upload logo IWA pronto
- Upload immagine inferiore generata con GPT Image
- Anteprima live
- Download PNG

## Sviluppo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
